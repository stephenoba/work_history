1. Javascript is a dynamically typed language. What does this mean?
This simply means that in JavaScript your code is flexible. Variables could be
used to hold different data types.

2. What is a primitive type? Explain each primitive type.
A primitive type is a type of data that is not an object and has no methods.

String: this is a series of characters which a written within quotes, single or double.
Number: This are numeric values and are written without any surrounding quotes.
Booleans: This is a primitive type in JavaScript that has just two values, either True or False
Undefined: This is a variable without a value.
Symbol: A symbol is a unique and immutable data type and may be used as an identifier for object properties

3. What are operators and operands?
Operators are the symbols that represent specific action, while Operands are the objects that are manipulated

4. In what order does operators get called?
Operators are called depending on their precedence and where they have equal
precedence they are evaluated according to their associativity.

5. What does it mean when a value is coerced?
It means converting a value from one type to another.

6. What is the difference between '==' and '==='?
== is an abstract equality comparison which performs a type conversion when comparing two things
while === is a strict equality comparison which doesn't perform a type conversion when comparing two things.
